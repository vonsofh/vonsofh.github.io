## Tentang Proyek

GAME PROXY adalah script proxy untuk game online yang memungkinkan pengiriman custom command dan bypass prompt. Dilengkapi Discord bot yang di-deploy ke server.

### Fitur Utama

- **Script proxy Lua** — Kirim custom command atau skip prompt dari game
- **Discord bot** — Bot yang di-upload ke server dan ditambahkan ke beberapa server Discord
- **Automasi** — Otomatisasi interaksi dengan game melalui proxy

### Arsitektur

Script proxy ditulis dalam Lua untuk intercepting dan manipulasi game traffic. Discord bot dibangun dengan Python/JavaScript dan di-deploy ke cloud server untuk digunakan di banyak server Discord.
