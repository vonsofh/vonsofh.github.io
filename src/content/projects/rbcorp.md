## Tentang Proyek

RBCORP.ID adalah website company profile untuk korporasi yang dibangun menggunakan WordPress. Admin bisa mengelola post dan seluruh konten website.

### Fitur Utama

- **CMS WordPress** — Kelola konten website dengan mudah menggunakan WordPress
- **Manajemen post** — Buat dan kelola artikel/berita perusahaan
- **Konten dinamis** — Admin bisa mengubah semua konten dari dashboard
- **Desain profesional** — Tampilan korporat yang bersih dan modern

### Arsitektur

Dibangun menggunakan WordPress dengan tema custom untuk tampilan korporat. Plugin WordPress digunakan untuk fitur tambahan seperti SEO dan contact form.
