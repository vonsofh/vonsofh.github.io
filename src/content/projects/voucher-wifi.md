## Tentang Proyek

Voucher WiFi adalah sistem penjualan voucher WiFi hotspot berbasis web. Pengguna bisa membeli voucher dan membayar melalui banyak channel pembayaran, terintegrasi langsung dengan perangkat Mikrotik.

### Fitur Utama

- **Pembelian voucher online** — Pengguna bisa beli voucher WiFi langsung dari website
- **Multi payment channel** — Berbagai opsi pembayaran via Tripay
- **Integrasi Mikrotik** — Auto-provisioning voucher ke perangkat Mikrotik
- **Desain simpel** — Menggunakan Bootstrap 5 yang clean dan user-friendly
- **Landing page hotspot** — Halaman login dan landing untuk RouterOS

### Arsitektur

Dibangun dengan Laravel dan Bootstrap 5. Terintegrasi dengan Tripay untuk multi-channel payment dan Mikrotik RouterOS API untuk auto-provisioning voucher WiFi ke perangkat jaringan.
