## Tentang Proyek

SIMPLE CRM adalah aplikasi Customer Relationship Management untuk mencatat data perusahaan dan karyawan. Dibangun dengan tampilan AdminLTE yang intuitif. **7 stars** di GitHub.

### Fitur Utama

- **CRUD perusahaan & karyawan** — Kelola data perusahaan dan karyawan dengan DataTable
- **Upload media** — Fitur upload file dan media
- **Multi-bahasa** — Dukungan beberapa bahasa
- **Email notifikasi** — Notifikasi email otomatis saat perusahaan baru terdaftar
- **JWT API** — API dengan autentikasi JSON Web Token
- **AdminLTE 3.1.0** — Dashboard modern dan responsif

### Arsitektur

Dibangun dengan Laravel menggunakan AdminLTE sebagai UI framework. JWT digunakan untuk API authentication, terintegrasi dengan Company App sebagai consumer. Email notification via Laravel Mail.
