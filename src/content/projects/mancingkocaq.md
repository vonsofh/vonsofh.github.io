## Tentang Proyek

MANCINGKOCAQ STORE adalah website e-commerce yang menjual berbagai produk game. Dilengkapi landing page menarik yang menampilkan produk unggulan.

### Fitur Utama

- **Landing page produk unggulan** — Tampilan menarik menampilkan featured products
- **Admin panel lengkap** — Kelola pelanggan, produk, keranjang, diskon, dan lainnya
- **OAuth Roblox** — Integrasi login menggunakan akun Roblox
- **Pembayaran QRIS dinamis** — Konversi manual QRIS Dynamic untuk pembayaran
- **Integrasi WhatsApp** — Notifikasi otomatis via WhatsApp

### Arsitektur

Dibangun dengan Laravel sebagai backend e-commerce fullstack. Sistem menggunakan OAuth V2 untuk autentikasi Roblox, QRIS Dynamic untuk pembayaran, dan WhatsApp API untuk notifikasi transaksi.
