## About the Project

VOUCHER WIFI is an e-commerce platform for selling WiFi hotspot vouchers. Integrates with Mikrotik routers for voucher generation and Tripay for payment processing with automated delivery.

### Key Features

- **Online voucher store** — Browse and purchase WiFi vouchers by duration/speed
- **Mikrotik integration** — Auto-generate hotspot vouchers from Mikrotik router
- **Tripay payment** — Multiple payment methods via Tripay gateway
- **Automated delivery** — Voucher code sent instantly after payment confirmation
- **Sales dashboard** — Revenue tracking and voucher usage analytics

### Architecture

Laravel storefront connected to Mikrotik RouterOS API for real-time voucher generation. Tripay webhook handles payment confirmation and triggers automated voucher delivery to customers.
