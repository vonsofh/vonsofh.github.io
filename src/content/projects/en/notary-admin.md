## About the Project

NOTARY ADMIN is a notary administration system for managing notarial deeds, client registry, and document workflows. Features automated deed numbering and comprehensive record keeping.

### Key Features

- **Auto deed numbering** — Sequential deed number generation per year/type
- **Client registry** — Complete client database with document history
- **Document management** — Digital storage and retrieval of notarial documents
- **Deed templates** — Pre-configured templates for common deed types
- **Transaction log** — Full audit trail of all notarial activities

### Architecture

Laravel application with automated deed numbering engine. MySQL handles relational data between clients, deeds, and documents. File storage system for digital document archival.
