## About the Project

LEARNING CMS is a learning management system with API login integrated with SMK Negeri 5. Manages teachers, students, and course content with role-based access control.

### Key Features

- **API login integration** — Authentication linked to SMK5 school system
- **Teacher management** — Course assignment, material upload, and grading
- **Student dashboard** — Access to learning materials and progress tracking
- **Course content** — Structured lesson modules with media support
- **Role-based access** — Separate interfaces for teachers and students

### Architecture

Laravel backend with API authentication bridge to SMK5's existing system. CMS manages course content, user roles, and learning materials. REST API enables integration with the school's infrastructure.
