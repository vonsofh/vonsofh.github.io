## About the Project

EDU INSENTIF is a teacher incentive management system featuring 8 distinct user roles. The system automates incentive calculations based on teaching performance data and predefined criteria.

### Key Features

- **8 user roles** — Granular access control from admin to teaching staff
- **Automated calculation** — Incentive amounts computed from performance metrics
- **Approval workflow** — Multi-level verification before incentive disbursement
- **Reporting** — Detailed incentive reports by period, school, and individual
- **Data import** — Bulk teacher data and performance metrics import

### Architecture

Built with Laravel featuring role-based middleware for 8 access levels. Complex incentive calculation engine processes multiple performance variables. MySQL stores hierarchical organizational and performance data.
