## About the Project

CLICKORAMA is a cross-platform photobooth system that connects an Electron-based desktop application with a Laravel admin panel. Photos are captured and synchronized in real-time for centralized management.

### Key Features

- **Dual platform** — Electron desktop app for photobooth operations, Laravel dashboard for administration
- **Real-time sync** — Photos automatically synced between desktop and admin panel
- **Session management** — Photo session tracking and management
- **Gallery system** — Organized photo galleries with filtering and download options

### Architecture

Desktop application built with ElectronJS handles photo capture and local processing. Laravel backend manages admin dashboard, data storage, and real-time synchronization between platforms via REST API.
