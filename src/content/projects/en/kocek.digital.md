## About the Project

KOCEK.DIGITAL is a digital finance platform that enables users to manage their balance, process transactions, and integrate payments with ease — for both personal and business use.

### Key Features

- **Balance management** — Easily and securely manage digital balances
- **Transaction history** — Real-time record of all transactions
- **Payment integration** — Support for various popular payment methods
- **Analytics dashboard** — Monitor income and expenses from a single view
- **Multi-user** — Supports both personal and business accounts

### Architecture

Built with Laravel as the main backend with a REST API layer. The system handles financial transactions with multi-layer security and supports integration with various payment gateways.
