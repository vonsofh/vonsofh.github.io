## About the Project

SKYPAY is a multi-service payment hub combined with network device management for Skynetwork ISP. Handles customer billing, payment processing via Tripay, and direct Mikrotik router control.

### Key Features

- **Payment hub** — Centralized payment processing with Tripay integration
- **Network device control** — Mikrotik API for router and bandwidth management
- **Customer billing** — Automated invoice generation and payment tracking
- **Multi-service** — Handles internet and other service payments
- **Admin dashboard** — Revenue analytics and device status monitoring

### Architecture

Laravel-based SaaS platform. Tripay payment gateway handles transaction processing. Mikrotik API integration enables direct network device control — user provisioning, bandwidth management, and device monitoring from a single dashboard.
