## About the Project

PASAR SAYUR POS is a multi-tenant point-of-sale system designed for vegetable markets. Desktop cashier built with Electron syncs with a Laravel cloud dashboard for centralized management across multiple market stalls.

### Key Features

- **Multi-tenant architecture** — Each market stall operates independently with isolated data
- **Desktop POS** — ElectronJS cashier app for fast offline-capable transactions
- **Cloud sync** — Real-time data synchronization between desktop and cloud admin
- **Sales analytics** — Comprehensive reports including daily/weekly/monthly summaries
- **Inventory management** — Stock tracking with low-stock alerts

### Architecture

Electron desktop app handles POS transactions locally for speed and reliability. Laravel cloud backend provides multi-tenant admin dashboard, analytics, and centralized data storage. REST API ensures real-time synchronization.
