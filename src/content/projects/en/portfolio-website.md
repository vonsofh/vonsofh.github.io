## Overview

This is my personal portfolio website — a blazing-fast static site built with Astro and Tailwind CSS, hosted for free on GitHub Pages.

## Problem

I needed a professional online presence to showcase projects, share blog posts, and give people an easy way to reach me. WordPress felt overkill, and I wanted full design control.

## Solution

Built a custom static site with:

- **Astro** as the framework (excellent performance + content collections)
- **Tailwind CSS** for utility-first styling
- **Markdown** for easily editable content files
- **GitHub Actions** for automated deployment

## Key Features

- Multi-page layout (Home, About, Projects, Experience, Blog, Contact)
- SEO optimized with Open Graph tags, sitemap, and RSS feed
- Responsive design that works on all devices
- Contact form using Formspree
- WhatsApp quick-chat link
- Indonesian and English language support
- Dark mode

## Lessons Learned

Building a portfolio from scratch gave me a deeper understanding of static site generation, content modeling, and modern CSS workflows. Astro's content collections made managing structured content incredibly easy without a database.
