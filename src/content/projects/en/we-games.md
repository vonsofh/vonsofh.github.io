## About the Project

WE GAMES PLAYSTATION is a billing system for PlayStation rental venues. Features IoT-based remote power control for each console, combined with food & beverage ordering and per-session payment tracking.

### Key Features

- **IoT remote control** — Turn PlayStation consoles on/off remotely via IoT integration
- **Session billing** — Automated time-based billing per console
- **F&B integration** — Food and beverage ordering linked to active sessions
- **Payment tracking** — Combined billing for gaming sessions and F&B orders
- **Admin dashboard** — Real-time console status and revenue monitoring

### Architecture

Laravel backend manages billing logic, session tracking, and order processing. IoT module communicates with physical power relays to control console power states. REST API connects the admin dashboard with hardware controllers.
