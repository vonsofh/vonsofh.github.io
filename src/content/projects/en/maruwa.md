## About the Project

PT MARUWA INDONESIA is a stock management system for PT Maruwa Indonesia. The system manages inventory flow with 3 distinct user roles: admin, purchasing, and warehouse.

### Key Features

- **3-role access** — Admin, purchasing, and warehouse with different permissions
- **Stock tracking** — Real-time inventory monitoring with item flow history
- **Purchase orders** — Purchasing role manages procurement and supplier data
- **Warehouse management** — Stock in/out recording and storage tracking
- **Reports & analytics** — Stock reports, movement history, and low-stock alerts

### Architecture

Built with Laravel and MySQL. Role-based access control separates admin, purchasing, and warehouse functions. Stock movement tracked with full audit trail.
