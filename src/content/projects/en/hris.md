## About the Project

HRIS is a Human Resource Information System with JWT-based authentication. Manages employee data, attendance tracking, leave requests, and payroll processing.

### Key Features

- **Employee management** — Complete employee profiles and organizational structure
- **Attendance tracking** — Daily check-in/check-out with overtime calculation
- **Leave management** — Leave request submission and approval workflow
- **Payroll** — Salary calculation with deductions and allowances
- **JWT authentication** — Secure API access with token-based auth

### Architecture

Laravel backend with JWT authentication for secure API access. REST API enables integration with front-end applications and mobile clients. MySQL database stores employee records and transaction data.
