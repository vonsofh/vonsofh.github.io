## About the Project

SMART COURIER is a courier delivery simulation implementing the A* pathfinding algorithm. Features interactive map visualization showing optimal route calculation for package deliveries.

### Key Features

- **A* pathfinding** — Optimized delivery route calculation
- **Interactive map** — Visual representation of the delivery grid
- **Route visualization** — Step-by-step animated path display
- **Multiple deliveries** — Handles multiple pickup and drop-off points

### Architecture

Pure JavaScript implementation with HTML Canvas for map rendering. A* algorithm calculates optimal paths considering obstacles and distance heuristics.
