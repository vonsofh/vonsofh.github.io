## About the Project

MANCINGKOCAQ is an e-commerce platform for gaming products such as top-ups, in-game items, and game-related merchandise. Features Roblox OAuth login and QRIS digital payment integration.

### Key Features

- **Roblox OAuth login** — Seamless authentication using Roblox accounts
- **QRIS payment** — Digital payment via QRIS for fast checkout
- **Product catalog** — Gaming products organized by game and category
- **Automated delivery** — Instant product delivery after payment confirmation
- **Order tracking** — Real-time order status and transaction history

### Architecture

Laravel handles the storefront, product management, and order processing. Roblox OAuth integration for user authentication. QRIS payment gateway for automated payment verification and product delivery.
