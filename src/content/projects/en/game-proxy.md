## About the Project

GAME PROXY is a gaming proxy server with integrated Discord bots. The Lua-based proxy handles game traffic, while Python and JavaScript Discord bots provide server management and player interaction features.

### Key Features

- **Lua proxy server** — Game traffic routing and management
- **Discord bot (Python)** — Server management commands and player stats
- **Discord bot (JavaScript)** — Additional bot features and integrations
- **Server management** — Remote game server control via Discord commands
- **Player tools** — In-game utilities accessible through bot commands

### Architecture

Lua proxy server handles game network traffic. Python and JavaScript bots connect to Discord API for server management features. Bots communicate with the game proxy for real-time server control.
