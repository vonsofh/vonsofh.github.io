## About the Project

E-PERPUSTAKAAN is a cross-platform library application connecting a mobile app with an admin dashboard through a single REST API database. Built over 4 months using Ionic/Angular for mobile.

### Key Features

- **Cross-platform mobile** — Mobile application using Ionic framework
- **Admin dashboard** — Management panel built with Laravel
- **REST API** — Unified database connection between platforms via REST API
- **Book catalog** — Search and browse the library collection
- **Digital borrowing** — Book lending system from the mobile app

### Architecture

Mobile frontend built with Ionic/Angular (TypeScript), connected to a Laravel backend via REST API. Separate admin panel uses Laravel Blade. Single MySQL database shared across all platforms.
