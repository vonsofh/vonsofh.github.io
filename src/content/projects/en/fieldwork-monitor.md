## About the Project

FIELDWORK MONITOR is a multi-tenant field reporting system designed for real-time monitoring and coordination of field operations teams.

### Key Features

- **Multi-tenant architecture** — Separate data isolation for each organization
- **Real-time task monitoring** — Live tracking of field team activities
- **Report management** — Field workers submit reports with photo attachments
- **Dashboard analytics** — Summary statistics and operational insights

### Architecture

Built with Laravel multi-tenant architecture. REST API serves the field worker mobile interface. Admin dashboard provides real-time monitoring and task assignment capabilities.
