## About the Project

SIMPATRI is a cross-platform application for reporting illegal workers in Batam. The Flutter mobile app enables citizens to capture evidence and send SOS alerts, while the Laravel backend handles report processing with role-based workflows.

### Key Features

- **SOS emergency button** — Quick alert system for urgent situations
- **Evidence capture** — Photo and document attachment for reports
- **Role-based processing** — Multi-level report verification and handling
- **Real-time notifications** — Status updates on submitted reports
- **Cross-platform** — Flutter mobile app connected to Laravel admin

### Architecture

Flutter mobile app communicates with Laravel backend via REST API. Role-based access control manages report flow from submission through verification to resolution. MySQL database with secure file storage for evidence.
