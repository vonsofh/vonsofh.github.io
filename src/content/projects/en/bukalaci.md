## About the Project

BUKALACI is the SaaS evolution of PasarSayur — a multi-tenant cloud platform for store management, product catalog, and web-based point-of-sale. Designed to be accessible for anyone who wants to open and manage a digital store.

### Key Features

- **Multi-tenant** — Each tenant gets an isolated store and dataset
- **Product management** — Easily add, edit, and organize product catalogs
- **Web-based POS** — Process sales transactions directly from the browser
- **Sales reports** — Automated daily, weekly, and monthly summaries
- **Stock management** — Monitor inventory levels with low-stock alerts

### Architecture

Built on Laravel with a multi-tenant architecture. Each tenant receives a separate subdomain or namespace with fully isolated data. REST API enables integration with third-party applications and tools.
