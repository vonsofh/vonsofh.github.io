## About the Project

BAIT MEDIA is a company profile website for a creative agency. Features portfolio showcase, Google OAuth authentication for team access, and an admin CMS for content management.

### Key Features

- **Portfolio showcase** — Visual gallery of creative works and projects
- **Google OAuth** — Team login via Google account authentication
- **Admin CMS** — Manage pages, portfolio items, and company information
- **Service pages** — Detailed service offerings and pricing
- **Contact system** — Inquiry form with email notifications

### Architecture

Built with Laravel and Tailwind CSS. Google OAuth provides secure team authentication. CMS allows non-technical staff to manage all website content.
