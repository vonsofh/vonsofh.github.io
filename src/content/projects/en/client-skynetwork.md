## About the Project

CLIENT SKYNETWORK is a client portal for an ISP company (Skynetwork). Features batch invoice generation, direct Mikrotik router integration, and a customer self-service dashboard.

### Key Features

- **Client self-service** — Customers view invoices, usage, and manage subscriptions
- **Batch invoicing** — Automated bulk invoice generation per billing cycle
- **Mikrotik integration** — Direct API connection to manage bandwidth and user profiles
- **Payment tracking** — Invoice payment status and history
- **Usage monitoring** — Bandwidth usage statistics for each client

### Architecture

Laravel handles client management, billing logic, and the admin panel. Mikrotik API integration automates bandwidth provisioning and user management. REST API serves the client-facing portal.
