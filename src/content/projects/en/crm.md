## About the Project

SIMPLE CRM is a Customer Relationship Management system with AdminLTE dashboard and multi-language support. Manages clients, deals, and sales pipeline.

### Key Features

- **Client management** — Contact database with interaction history
- **Deal pipeline** — Visual sales pipeline with stage tracking
- **Multi-language** — Interface available in multiple languages
- **AdminLTE dashboard** — Clean admin interface with widgets and charts
- **Activity tracking** — Log calls, meetings, and follow-ups per client

### Architecture

Laravel with AdminLTE template for the admin dashboard. Multi-language support via localization files. Pipeline engine tracks deal stages with automated status transitions.
