## About the Project

GOLDENSCHOOL BATAM is a landing page website for Golden School Batam. Features an admin panel for controlling all frontend content and school information display.

### Key Features

- **Informative landing page** — Complete information about Golden School Batam
- **Admin panel** — Backend administration for frontend content control
- **Dynamic content** — All content editable from the admin dashboard
- **Registration info** — New student enrollment information portal

### Architecture

Built with PHP and NodeJS for select backend features. Landing page displays school information that can be edited from the admin panel.
