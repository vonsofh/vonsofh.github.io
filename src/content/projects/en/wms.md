## About the Project

WMS SKYNETWORK is a WiFi Management SaaS platform that integrates with Mikrotik routers for hotspot management. Features automated voucher generation and multi-location support.

### Key Features

- **Mikrotik API integration** — Direct control of hotspot user profiles and bandwidth
- **Automated voucher generation** — Bulk WiFi voucher creation with configurable profiles
- **Multi-location** — Manage multiple hotspot locations from a single dashboard
- **Usage analytics** — Bandwidth usage and session statistics per location
- **User management** — Hotspot user CRUD with profile assignment

### Architecture

Laravel SaaS platform with Mikrotik RouterOS API integration. Each location connects to its own Mikrotik router for independent hotspot management. Centralized dashboard provides cross-location analytics.
