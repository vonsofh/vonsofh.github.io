## About the Project

THE SPACE EXPLORER is a space exploration video game that was successfully presented as a demo. The project was completed in approximately 3 months.

### Key Features

- **Level selection** — Multiple levels with varying difficulty
- **Game controllers** — Responsive character movement and controls
- **Game algorithms** — Core game logic including collision detection and scoring
- **Demo presentation** — Successfully presented as a complete playable demo

### Architecture

Built with C# using a game engine for 2D rendering. Controllers, game algorithms, and level design implemented from scratch.
