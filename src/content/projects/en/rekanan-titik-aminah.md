## About the Project

REKANAN TITIK AMINAH is a partner website for a notary office, built with Laravel Voyager CMS. Features partner directory, service listings, and client inquiry management.

### Key Features

- **Voyager CMS** — Admin panel powered by Laravel Voyager for easy content management
- **Partner directory** — Affiliated partner listings and profiles
- **Service catalog** — Notarial service descriptions and requirements
- **Client inquiry** — Contact form with inquiry routing
- **SEO optimized** — Clean URLs and meta tag management

### Architecture

Built on Laravel with Voyager CMS for admin functionality. Content management allows non-technical staff to update all website sections. Responsive design optimized for client access.
