## Tentang Proyek

TERRACE BIKER INVENTORY adalah sistem inventaris berbasis web untuk toko Terrace Biker. Mengelola seluruh data barang yang diinput oleh admin.

### Fitur Utama

- **Manajemen barang** — Input dan kelola data barang oleh admin
- **Log barang masuk/keluar** — Lacak pergerakan stok in/out
- **Dashboard admin** — Pantau status inventaris dari satu dashboard
- **Laporan stok** — Rekap stok per periode

### Arsitektur

Dibangun dengan Laravel dan Blade template engine. Sistem menggunakan log-based tracking untuk mencatat setiap pergerakan barang masuk dan keluar.
