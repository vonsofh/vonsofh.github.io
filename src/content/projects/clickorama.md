## Tentang Proyek

CLICKORAMA.ID adalah aplikasi desktop dan web untuk sinkronisasi photobooth. Sistem ini mengelola seluruh alur kerja photobooth mulai dari event, booth, hingga distribusi foto ke pelanggan.

### Fitur Utama

- **Sinkronisasi photobooth** — Aplikasi desktop tersinkronisasi dengan server web secara real-time
- **Manajemen event** — Fitur acara seperti pernikahan, ulang tahun, dan anniversary yang bisa digunakan pelanggan
- **Cloud share via QR** — Di akhir sesi, pelanggan scan QR untuk mengunduh semua foto langsung dari cloud
- **Admin panel** — Kelola seluruh organisasi, booth, event, dan konfigurasi dari halaman admin
- **Multi-organization** — Mendukung banyak organisasi dengan booth dan event masing-masing

### Arsitektur

Dibangun dengan Laravel sebagai backend API dan ElectronJS untuk aplikasi desktop. Frontend menggunakan jQuery untuk interaktivitas, sementara cloud storage terintegrasi untuk distribusi foto otomatis via QR code.
