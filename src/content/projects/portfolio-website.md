## Ringkasan

Ini adalah website portofolio pribadi saya — situs statis yang sangat cepat, dibangun dengan Astro dan Tailwind CSS, di-hosting gratis di GitHub Pages.

## Permasalahan

Saya membutuhkan kehadiran online yang profesional untuk menampilkan proyek-proyek, membagikan tulisan blog, dan menyediakan cara mudah bagi orang untuk menghubungi saya. WordPress terasa terlalu berlebihan, dan saya ingin kontrol desain sepenuhnya.

## Solusi

Membangun situs statis custom dengan:

- **Astro** sebagai framework (performa luar biasa + content collections)
- **Tailwind CSS** untuk styling berbasis utility
- **Markdown** untuk file konten yang mudah diedit
- **GitHub Actions** untuk deployment otomatis

## Fitur Utama

- Layout multi-halaman (Beranda, Tentang, Proyek, Pengalaman, Blog, Kontak)
- SEO optimal dengan Open Graph tags, sitemap, dan RSS feed
- Desain responsif yang bekerja di semua perangkat
- Form kontak menggunakan Formspree
- Link chat cepat WhatsApp
- Dukungan bahasa Indonesia dan Inggris
- Mode gelap (dark mode)

## Pelajaran yang Didapat

Membangun portofolio dari nol memberikan pemahaman lebih mendalam tentang static site generation, pemodelan konten, dan workflow CSS modern. Content collections dari Astro membuat pengelolaan konten terstruktur sangat mudah tanpa database.
