## Tentang Proyek

E-PERPUSTAKAAN adalah aplikasi perpustakaan cross-platform yang menghubungkan mobile app dengan admin dashboard menggunakan satu database melalui REST API. Dibangun dalam 4 bulan.

### Fitur Utama

- **Cross-platform mobile** — Aplikasi mobile menggunakan Ionic framework
- **Admin dashboard** — Halaman admin menggunakan Laravel
- **REST API** — Hubungan antar platform menggunakan REST API dengan satu database
- **Katalog buku** — Pencarian dan browse koleksi perpustakaan
- **Peminjaman digital** — Sistem peminjaman buku dari mobile app

### Arsitektur

Frontend mobile dibangun dengan Ionic/Angular (TypeScript), terhubung ke backend Laravel melalui REST API. Admin panel terpisah menggunakan Laravel Blade. Satu database MySQL digunakan untuk semua platform.
