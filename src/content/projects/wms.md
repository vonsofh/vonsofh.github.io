## Tentang Proyek

WMS SKYNETWORK.ID adalah SaaS WiFi Management Service yang memungkinkan pengelolaan perangkat Mikrotik melalui API. Sistem bisa generate voucher WiFi dan berkomunikasi langsung dengan perangkat jaringan.

### Fitur Utama

- **Manajemen Mikrotik via API** — Kontrol perangkat Mikrotik langsung dari dashboard
- **Generate voucher** — Buat voucher WiFi otomatis
- **Validasi perangkat** — Berkomunikasi dan memvalidasi ke perangkat jaringan
- **SaaS multi-tenant** — Mendukung banyak operator/ISP
- **Admin dashboard** — Kelola seluruh perangkat dan voucher dari satu tempat

### Arsitektur

Dibangun dengan Laravel, terintegrasi dengan Mikrotik RouterOS API untuk manajemen perangkat secara remote. Arsitektur SaaS memungkinkan banyak operator ISP menggunakan satu platform untuk mengelola jaringan WiFi.
