## Tentang Proyek

CLIENT.SKYNETWORK.ID adalah portal client untuk ISP (Internet Service Provider) berbasis Laravel. Sistem mengelola langganan pelanggan, pembuatan invoice batch, dan informasi layanan internet.

### Fitur Utama

- **Manajemen langganan** — Kelola seluruh langganan pelanggan internet
- **Batch invoice** — Buat invoice massal untuk semua pelanggan sekaligus
- **Dashboard admin** — Kelola seluruh konten, pelanggan, dan billing
- **Integrasi Mikrotik** — Terhubung dengan perangkat jaringan untuk monitoring

### Arsitektur

Dibangun dengan Laravel sebagai portal client ISP. Terintegrasi dengan Mikrotik API untuk sinkronisasi data layanan internet pelanggan. Admin bisa mengelola batch billing dan monitoring langganan.
