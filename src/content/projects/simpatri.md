## Tentang Proyek

SIMPATRI (Sistem Informasi Patriot) adalah aplikasi untuk melaporkan orang yang berniat bekerja secara ilegal ke luar negeri. Sistem terdiri dari web admin dan aplikasi mobile untuk pelaporan langsung.

### Fitur Utama

- **Pelaporan publik** — Masyarakat umum bisa melaporkan dan menggunakan aplikasi
- **Aplikasi mobile** — Fitur SOS dan pelaporan kasus langsung dari lokasi dengan live tracking
- **Admin panel UI/UX** — Dashboard admin yang intuitif untuk mengendalikan semua aktivitas
- **Live tracking** — Pemantauan lokasi secara real-time saat pelaporan di lapangan
- **Manajemen kasus** — Kelola dan pantau seluruh laporan dari satu dashboard

### Arsitektur

Frontend mobile dibangun dengan Flutter untuk cross-platform (Android & iOS), terhubung ke backend Laravel via REST API. Admin panel berbasis web dengan UI/UX yang dirancang untuk kemudahan operasional.
