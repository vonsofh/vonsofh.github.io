## Tentang Proyek

REKANAN TITIK AMINAH adalah website rekanan untuk Notaris PPAT Titik Aminah. Partner bisa melihat daftar klien dan detail rekanan dari satu portal. **5 stars** di GitHub.

### Fitur Utama

- **Daftar klien** — Buat dan lihat daftar klien dengan mudah
- **Detail rekanan** — Partner Titik Aminah bisa melihat detail klien
- **Admin CMS** — Kelola konten menggunakan Voyager admin panel
- **Portal partner** — Akses khusus untuk rekanan notaris

### Arsitektur

Dibangun dengan Laravel dan Voyager sebagai admin panel CMS. Voyager menyediakan interface CRUD untuk manajemen data klien dan rekanan tanpa perlu coding tambahan.
