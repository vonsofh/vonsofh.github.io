## Tentang Proyek

WE GAMES PLAYSTATION adalah sistem billing dan manajemen untuk tempat rental PlayStation. Terintegrasi dengan IoT remote untuk kontrol TV otomatis.

### Fitur Utama

- **Manajemen billing** — Kelola durasi bermain dan perhitungan biaya otomatis
- **Order F&B** — Pelanggan bisa memesan makanan dan minuman dari sistem
- **IoT remote integration** — Kontrol otomatis untuk menyalakan/mematikan TV via IoT
- **Landing page real-time** — Pelanggan bisa melihat ketersediaan perangkat secara langsung (auto update)
- **Admin panel** — Kelola seluruh konten, perangkat, dan order dari dashboard

### Arsitektur

Dibangun dengan Laravel fullstack, terintegrasi dengan IoT device untuk remote control TV. Landing page menggunakan polling/websocket untuk update real-time ketersediaan perangkat.
